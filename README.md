# Customer Churn Prediction using ANN

This project predicts customer churn using Artificial Neural Network (ANN).

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras

## Features
- Data cleaning
- Feature engineering
- Data scaling
- ANN model training
- Accuracy evaluation
- Confusion matrix

## Accuracy
Model accuracy achieved: 78%
